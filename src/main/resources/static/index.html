API接口信息

  • 简单聊天: GET /api/ai/simple-chat?message=xxx
  • 流式聊天: GET /api/ai/stream-chat?message=xxx (SSE)
  • 带历史流式聊天: GET /api/ai/stream-chat-with-history?message=xxx (SSE)
  • React Agent: GET /api/ai/react-agent?message=xxx (SSE)
  • 合同审核: GET /api/ai/contract-review?contractText=xxx (SSE)
  • 技能合同审核: GET /api/ai/skill-contract-review?contractText=xxx (SSE)
  • 技能列表: GET /api/ai/skills
  • 执行技能: POST /api/execute-skill

流式响应使用Server-Sent Events (SSE)，可以实时接收AI的回复。
